# Xiaomi MiMo Orbit Application Brief

DevOrbit is a developer-tool project designed to showcase Agent-style software engineering workflows:

- repository-aware scanning
- multi-agent review/fix/test/presentation roles
- deterministic mock provider for offline execution
- provider interface reserved for OpenAI-compatible and MiMo-compatible APIs
- Markdown reports and terminal output suitable for demos

## Demo Claim Boundary

All generated reports, metrics, and logs in this repository are demo artifacts unless otherwise documented with real evidence. The project must not fabricate users, deployments, revenue, token usage, or benchmark performance.
